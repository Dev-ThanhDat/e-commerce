# E-Commerce website

> An E-Commerce platform allows users to shop online.

> Members: 1 (personal project)

### Technology used:

- ReactJS, Tailwind CSS, Ant Design, Redux Toolkit, React Hook Form.
- NodeJS, ExpressJS, MongoDB, Mongoose, Cloudinary, JWT.

### Describe:

- User: Search, filter products, add a shopping cart, buy, evaluate products, view and track orders.

- Admin: CRUD Manager user, products, orders, categories, discount codes, colors and articles.

- Integrated Restful API to manipulate data. Apply JWT to decentralize and authenticate.

- Integrated Cloudinary and image processing before uploading.

- Manage data with Redux Toolkit and use Lazy Loading to improve performance.

### Note:
