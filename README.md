# E-Commerce website

> This is the E-commerce project for fullstack.

### Technology used:

- ReactJS, Tailwind CSS, Ant Design, Redux Toolkit, React Hook Form.
- NodeJS, ExpressJS, MongoDB, Mongoose, Cloudinary.

### Function:

- Home shows all outstanding products and articles.

- The store page displays all products and can be searched by name, price, and product type as well as the amount of evaluation.

- The article page displays all posts and can be searched by name and type of article.

- And there are buying orders, favorite products and shopping carts.

- Product details and articles, in product details, you can choose to buy products and evaluate products.

- Admin page can manage everything of the website such as creating and editing, deleting users, products, discount codes, articles, categories and orders.
